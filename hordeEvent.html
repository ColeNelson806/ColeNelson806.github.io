Horde Event


The Horde Event plugin is a plugin I am creating with the idea of giving a little bit more spice to typical vanilla gameplay. It works by checking to see if a player killed a zombie, and if they did it will then have a chance to spawn a pack of mobs nearby. The chance, difficulty of the pack, along with amount of mobs will all be able to be customized using in game commands.

Commands
/heal Heals the user to full health
/feed Fills the user's hunger and saturation
